## Dependencies

- Node 5.10.x or above
- express: ^4.13.4

- bcrypt: ^1.0.2
- body-parser: ^1.15.2
- chance: ^1.0.2
- cookie-session: ^2.0.0
- md5: ^2.1.0
- mongodb: ^2.2.26


## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm run local` command.
